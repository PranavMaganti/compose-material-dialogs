MaterialDialog

class MaterialDialog(autoDismiss: Boolean, onCloseRequest: (MaterialDialog) -> Unit)
Content copied to clipboard

The MaterialDialog class is used to build and display a dialog using both pre-made and custom views

Parameters

autoDismiss

when true the dialog will be automatically dismissed when a positive or negative button is pressed

onCloseRequest

a callback for when the user tries to exit the dialog by clicking outside the dialog. This callback takes the current MaterialDialog as a parameter to allow for the hide method of the dialog to be called if required. By default this callback hides the dialog.
